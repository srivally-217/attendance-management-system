# Smart Attendance System

A Java-based Smart Attendance System built using Swing GUI.  
This application allows teachers or users to manage student attendance through an easy-to-use graphical interface.

---

## Features

- Add and remove students
- Mark selected students as Present or Absent
- Mark all students Present or reset to Absent
- Save attendance data to CSV file
- Load attendance data from CSV
- Export attendance reports
- Simple and user-friendly GUI

---

## Technologies Used

- Java
- Java Swing (GUI)
- File Handling (CSV)
- JTable and DefaultTableModel
